# CodeLeap FrontEnd

Frontend application built with React, TypeScript, and Vite for the CodeLeap challenge.

## Deploy

The application is live at:

https://vercel.com/renato-chagas-de-moraes-projects/code-leap-front-end

## Technologies

- React
- TypeScript
- Vite
- Axios
- React Router

## Running Locally

1. Clone the repository:

```bash
git clone https://github.com/renato-chagas/CodeLeap-FrontEnd.git
```

2. Go to the project folder:

```bash
cd CodeLeap-FrontEnd
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Available Scripts

- `npm run dev`: starts the project in development mode
- `npm run build`: generates the production build
- `npm run preview`: runs the build locally
- `npm run lint`: runs lint checks
